# Changelog

Todos los cambios importantes de este proyecto se documentarán en este archivo.

El formato está inspirado en [Keep a Changelog](https://keepachangelog.com/)
y este proyecto sigue un versionado simple.

---

## [0.2.4] 2026-01-13

### Changed
- Pequeñas correcciones de errores
- Se quitaron los comments dentro del archivo .json

### Notes
- Estos errores surgieron devido al cambio de nombre de la extension
- Se quitaron los comments devido que el marketplace no los acepta bien

## [0.2.3] 2026-01-13

### Changed
- Cambio del nombre de la extension.

### Notes
-Se hizo un cambio de nombre con el fin de ser mas localizable ya que tenia un nombre muy generico y usado en marketplace

---

## [0.2.2] 2026-01-13

### Added
- Se agrego la extencion a marketplace.

### Notes
-Con este cambio ya no se requiere una instalacion manual desde github, aunque sigue siendo una opcion.

---

## [0.2.1] - 2026-01-13

### Changed
- Cambio de color de los comentarios

### Notes
- La decisión de este cambio fue mejorar la coherencia de color e igualar el diseño entre los distintos lenguajes.

---

## [0.2.0] - 2026-01-12

### Changed
- Rediseño completo de la paleta de colores para HTML
- Ajustes en la paleta de colores de CSS

### Notes
- El estilo inicial de HTML no me convencía, por lo que fue rediseñado por completo.
- La paleta de CSS se ajustó para encajar mejor con el nuevo estilo de HTML.

---

## [0.1.1] - 2026-01-10

### Added
- Soporte para JSON

### Changed
- Ajustes en la paleta de colores con estilo similar a CSS
- Pequeños cambios en la paleta de:
  - CSS
  - JavaScript

### Notes
- Estos pequeños cambios facilitan la legibilidad del código.

---

## [0.1.0] - 2026-01-09

### Added
- Primera versión pública del tema **Fresh**
- Paleta de colores oscura, limpia y minimalista
- Soporte inicial para:
  - HTML
  - CSS
  - JavaScript
- Estilos optimizados para sesiones largas de programación

### Notes
- Esta es una versión temprana del tema.
- Es posible que algunos lenguajes o elementos de la UI aún no estén completamente pulidos.
